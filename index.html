hobby_site

  • Rails 4
  • Ruby 2.2
  • Bootstrap 3

VocabTester

This is a site that I made for fun to help me learn a new language. You register, create lists of words that you want to learn, and can take a test. The app keeps track of how many times you've gotten a word right or wrong.

Some tech this app uses

  • MongoDB 3
  • NodeJS
  • ExpressJS
  • Mongoose
  • AngularJS 1
  • Grunt
  • SASS
  • Bootstrap
